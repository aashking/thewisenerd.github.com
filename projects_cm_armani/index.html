[ROM][4.4.4][WEEKLIES] CyanogenMod 11.0 for Xiaomi Redmi 1S (armani)

Published October 18th 2014

CyanogenMod is a free, community built, aftermarket firmware distribution of Android 4.4.4 (KitKat), which is designed to increase performance and reliability over stock Android for your device.

As we got CM11.0 to beta-state fast, I decided to run weekly builds of it.

What are Weekly builds?

CM11.0 builds of the latest CyanogenMod sources from github, and latest armani-dev sources from GitHub so you always have latest CM features and fixes, and latest changes made by armani-dev.

New version will come out every week, most often during the weekends.

CyanogenMod is based on the Android Open Source Project with extra contributions from many people within the Android community. It can be used without any need to have any Google application installed. Linked below is a package that has come from another Android project that restore the Google parts. CyanogenMod does still include various hardware-specific code, which is also slowly being open-sourced anyway.

All the source code for CyanogenMod is available in the CyanogenMod Github repo. And if you would like to contribute to CyanogenMod, please visit their Gerrit Code Review.

Installation

  • Flash official TWRP recovery (http://en.miui.com/thread-29291-1-1.html)
  • Reboot into recovery
  • wipe data/factory reset + wipe cache partition
  • Install the ROM from internal memory/external sdcard from TWRP
  • optionally install the Google Apps (support regarding Google Apps will not be provided by OP)

FAQ

q: this error occours, will it be fixed with the next release?

a: not until I do not get the magic ball which can repair errors without logcat/kmsg.

q: Where are "Developer options" and "Performance" in Settings?

a: you must press the "Build number" 7 times in the "About phone"

q: I have a gray dot on top every time I enter the PIN. Is it normal?

upstream derp. doesn't affect functionality.

q: I do not sound the calls I get from SIM 2

a: upstream derp. second sim ringtone isn't set by default.
you must configure it.

  • open dialer
  • open menu -> settings -> subscription settings
  • switch to second tab (with your carrier name) set ringtone

q: is the volume is lower than in MIUI?

no, is the same.

q: Can I install apps from MIUI to CyanogenMod?

Most likely no and if you want to discuss please open another thread.

q: Could you add a feature that is present in another rom?

no; only features present in official CM.
